# We welcome your feedback

In this folder we have all our real estate core documents. It is divided into 2 main folders:
The core folder which contains all logical entities. It is also used for the entity navigator.
The resolved folder is created from the logical entities which resides under the core folder and contains all physical entities.
To visualize these entities please use the [CDM entity navigator](https://microsoft.github.io/CDM/SchemaViz.html).
For any further information please refer to the [Microsoft CDM](https://github.com/microsoft/CDM).

# How to use the Microsoft CDM entity navigator
1. Clone ibpdi cdm repository 
2. Launch the [Visual Entity Navigator](https://microsoft.github.io/CDM/)
3. Naviage to Schema Documents foler and upload in visual entity navigator for interactively exploring entities, entity extensions/inheritance, attributes, and relationships
